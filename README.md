# sound-changer

Phonological sound changer written in Crystal.
Accepts _n_ sound change rules and applies them to _m_ words.

## Installation

Add this to your application's `shard.yml`:

```yaml
dependencies:
  sound-changer:
    github: ceigey/sound-changer
```

## Usage

